# BRF Task Notifier

אפליקציית BRF Task Notifier היא לוח משימות משפחתי בעברית (RTL) המאפשר לתאם משימות, להוסיף תזכורות חכמות ולנהל את יומן המשפחה מכל מכשיר.

## 🚀 התקנה והפעלה
1. שיבטו את המאגר:
   ```bash
   git clone https://github.com/<your-username>/BRF.git
   ```
2. פתחו את התיקייה ופתחו את הקובץ `index.html` בדפדפן מודרני (Chrome, Edge, Safari).
3. להפעלה חיה מומלץ להשתמש ב-Live Server או לפרוס ב-GitHub Pages.

## ✨ תכונות עיקריות
- ניהול אנשים ומשימות עם תמיכה מלאה ב-RTL.
- שמירת נתונים אוטומטית באמצעות `localStorage`.
- תזכורות מבוססות Notification API עם צלמית PWA מותאמת.
- משימות חוזרות (יומי / שבועי) נוצרות מחדש לאחר מועד היעד.
- ממשק רספונסיבי המתאים לנייד, טאבלט ושולחן עבודה.

## 🔔 מערכת התראות
- בקשת הרשאת התראות עם הפעלת האפליקציה.
- בדיקת תזכורות אוטומטית כל 60 שניות, עם אפשרות "בדיקה עכשיו" ידנית.
- התראות מוצגות עם שם האדם והזמן בפורמט עברי.

## 📱 התקנה כ-PWA
1. פתחו את האפליקציה בדפדפן במכשיר נייד.
2. לחצו על "הוסף למסך הבית" (Chrome/Android) או שתפו → "הוסף למסך הבית" (Safari/iOS).
3. לאחר ההתקנה האפליקציה תפעל במצב מסך מלא עם סמל מותאם.

## 🌐 תאימות דפדפנים
- Android Chrome, Samsung Internet – תמיכה מלאה בהתראות ו-PWA.
- iOS Safari – נדרש להוסיף למסך הבית לקבלת התראות.
- דפדפני שולחן עבודה מודרניים (Chrome/Edge) תומכים בעבודה מלאה.
